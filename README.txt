Put your real White Paper file in this folder.

1. Name it exactly:   whitepaper-agecoin.pdf
   (If you'd rather link a Word file, name it whitepaper-agecoin.docx and
   update the href on the "Download White Paper" button in index.html.)

2. Upload this whole "agecoin" folder — including this assets folder — to
   your web host / server for agecoin.org.

That's it. The "Download White Paper (PDF)" button on the site already
points to assets/whitepaper-agecoin.pdf, so it will start working for every
visitor as soon as the real file is here.

The "Developer Console" button (bottom-right corner of the site) lets you
preview a PDF/Word file locally in your own browser before you upload it —
that preview is session-only and is not visible to site visitors.
